Overview

fixes:

  • autolinked email addresses that are actually references to users: user:<a href=“user@example.orguser@example.org”>user@example.org> @<a href=“user@example.orguser@example.org”>user@example.org>

  • autolinked hi res image names that look like email addresses: <a href=“printscreen@2x.pngprintscreen@2x.png”>printscreen@2x.png>

def call
  doc.search("a").each do |node|
    unless (url = node['href']) && url.starts_with?('mailto:')
      next
    end

    if ((p = node.previous) && p.text? &&
        p.text =~(USER_LINK_PREFIX)) ||
       (node.text =~ HIRES_IMAGE)

      node.replace node.text
    end
  end
  doc
end
